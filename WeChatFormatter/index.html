使用帮助

这是一款纯前端的微信公众号 Markdown 排版工具：左侧写 Markdown，点「一键排版」后右侧生成可直接粘贴到公众号的样式。

开始排版

  • 在左侧输入或粘贴 Markdown，也可点「导入」载入 .md 文件，或直接拖入文件。
  • 点击「一键排版」或按 Ctrl+S，将内容渲染为带内联样式的公众号 HTML。
  • 排版后点「复制到公众号」，粘贴到公众号后台即可保留样式。

样式与模板

  • 工具栏可调整 H1~H4 颜色/字号、表头色、字体、段间距、首行缩进。
  • 「样式模板」提供摸鱼绿、红白、石墨极简等多套公众号主题，一键切换。
  • 需要精细控制时可打开「CSS」自定义样式。

编辑与预览

  • 右侧预览区可直接微调文字；支持撤销/重做（Ctrl+Z / Ctrl+Y）。
  • 支持粘贴图片、粘贴 Excel 表格、插入公式与目录。
  • 公式支持 $...$ 行内与 $$...$$ 块级（一键排版后渲染为图片）。
  • 草稿会自动保存到浏览器本地，也可在「草稿」里管理多份内容。

导出与 AI

  • 可导出 Markdown、HTML、长图；也可复制纯文本。
  • 配置 DeepSeek API Key 后，可用 AI 助手做润色、扩写、缩写、翻译。可点击下方「API 配置」填写或修改 Key / 模型 / 接口。

常用快捷键：Ctrl+S 一键排版，Ctrl+F 搜索替换，? 快捷键面板，Esc 关闭弹窗。
