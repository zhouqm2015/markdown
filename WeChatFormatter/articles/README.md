# 公众号排版工具
写公众号最痛苦的，不是选题，不是写内容，而是排版。

在公众号后台改了无数遍：文字贴着边，代码块丑得没法看，表格一粘就碎，图片对不齐。折腾半小时排版，阅读量还没起来，心先累了。

用过各种在线工具：要么收费，要么一天只能免费用几次；改完还得再复制到后台微调。而且多数依赖联网，没网就干瞪眼。

我想要的很简单：**写 Markdown，一键变成能直接粘贴的公众号文章。本地能用，能粘图片，代码块好看，表格规整。**

找了一圈不完全合心意，干脆自己做了一个——**WeChatFormatter**。

在线体验：[https://www.zybkpro.top/WeChatFormatter/](https://www.zybkpro.top/WeChatFormatter/)

## 它具体解决什么问题

### 1. Markdown 一键转公众号格式

左边写 Markdown，点「一键排版」或按 `Ctrl+S`，右边生成**带内联样式**的 HTML。

公众号编辑器会丢掉 `class`，但一般不会拆掉标签上的 `style`。所以样式写在内联里，粘贴后更稳。

支持常见 GFM：标题、列表、表格、代码块、引用、加粗、删除线、行内代码，以及高亮 `==文字==`、着色 `{#E53E3E}文字{/#}` 等扩展。标题带左边框色块、代码块深色等宽、表格斑马纹圆角、引用灰底蓝边——都是公众号里常见的观感，开箱即用。

![一键排版效果](../README.assets/01.jpg)

### 2. 直接粘贴图片

截图、设计稿导出图，在编辑区 `Ctrl+V` 即可插入，自动生成 `![](pasted:N)` 并渲染到预览。不必先上图床再写链接。

导出 Markdown 时，粘贴图可打成 base64，换设备打开也能看到。

### 3. 粘贴 Excel 表格

从 Excel 复制单元格，粘到编辑区，自动转成 Markdown 管道表格。表头颜色还能在工具栏单独调。

### 4. 代码块友好

技术文最怕代码糊成一团。这里代码块用深色底 + 等宽字；行内代码浅灰底红字，正文里一眼能认出。

### 5. 主题模板与样式定制

- H1～H4 颜色 / 字号独立可调，表头色、字体、段间距、首行缩进都在工具栏
- 「样式模板」一键切换：摸鱼绿、红白、石墨极简、留白禅意、摸鱼票据、橄榄手记，以及 ColorUI、科技感、中华古典、水墨留白、赛博霓虹、暗黑等
- 「CSS」面板可写自定义样式，并内置更多配色预设（简约留白、暖阳橙调、森林绿意……）
- 当前配置可保存为自定义模板，也支持导入 / 导出 JSON，方便系列文统一风格

### 6. 复制到公众号，而不是「复制一段 HTML 自己碰运气」

点「复制到公众号」会做兼容处理：补全结构、烘焙计算后的内联样式、`div` 转成更稳妥的 `section`。再到后台 `Ctrl/⌘+V` 即可。需要纯文字时用「纯文本」。

### 7. 藏在细节里的辅助能力

- 行号、草稿自动保存、多草稿管理
- 目录：按 H1～H4 生成可点击目录
- 排版后注入字数 / 段落 / 预计阅读时间
- 历史版本：每次排版可留快照，最多约 20 条可回溯
- 常见错别字修正、敏感词警告
- 预览区可直接改字；`Ctrl+Z` / `Ctrl+Y` 撤销重做；`Ctrl+F` 搜索替换
- 手机预览（约 375px）、深浅色主题、全屏编辑、导出 MD / HTML / 长图
- 首次打开有使用帮助；页头「帮助」、`?` 快捷键面板随时可看

![目录生成](../README.assets/02.jpg)

### 8. 可选的 AI 助手（不挡本地主流程）

一键排版默认**纯本地**，不配 Key 也能完整用。

需要润色 / 扩写 / 缩写 / 翻译时，点页头「API 配置」填入兼容 OpenAI Chat Completions 的 Key（如 DeepSeek），再打开「AI 助手」。密钥只存在本机 `localStorage`，不写进源码。

### 9. 丰富的内置样式模板

工具栏「样式模板」一键切换，会同步标题色、字体、段距与 CSS：

**公众号主题**

- **摸鱼绿**：emerald 杂志风，适合教程、测评、清单
- **红白色系**：正红编辑风，适合深度观点、力量感文风
- **石墨极简风**：全灰阶细线留白，适合设计 / 科技评论
- **留白禅意风**：墨绿衬线，适合随笔、慢读
- **摸鱼票据风**：米黄硬阴影撕票感，适合测评对比
- **橄榄手记**：内刊墨黑 + 橙强调，适合案例复盘

**其他内置风格**

- **ColorUI**：活力圆角、渐变标题
- **科技感**：电光青、深色代码块
- **中华古典**：朱红墨色、楷体宣纸感
- **水墨留白**：黑白灰宋体，书卷气
- **赛博霓虹**：紫粉霓虹、暗夜科技
- **暗黑风格**：深色背景、冷蓝高亮

另外在「CSS」面板里还有简约留白、暖阳橙调、森林绿意、暮色紫调、海洋蓝调、国风典雅、柔和莫兰迪、代码舒适等配色预设。当前配置可另存为「我的模板」，也支持导入 / 导出 JSON。

## 和「又一个在线排版站」差在哪

| ` | 常见在线工具 | WeChatFormatter |
|--|--|--|
| 联网 | 多数必须在线 | 排版主流程纯本地 |
| 费用 | 常有次数墙 / 会员 | 打开即用 |
| 粘贴 | 有的只给 HTML，后台还要再调 | 针对公众号做复制兼容 |
| 密钥 | 常绑账号 | AI 可选，弹窗配置，存在本机 |
| 部署 | 只能用别人的站 | 静态目录，可自己托管 |

它解决的是一件很具体的事：**让 Markdown 到公众号，少折腾、可离线、样式还能留下来。**

写得好，排版也该足够简单。
