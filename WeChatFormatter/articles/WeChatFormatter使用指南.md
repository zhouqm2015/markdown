# WeChatFormatter 使用指南



---

一款为内容创作者打造的 **Markdown → 微信公众号** 排版工具。  
纯前端、本地优先：打开网页即可用，排版结果可一键复制到公众号后台。

在线体验：https://www.zybkpro.top/WeChatFormatter/

本文将带你从基础语法到高级能力，快速上手 WeChatFormatter。

![WeChatFormatter 主界面](../README.assets/image-20260617174929540.png)

## 编辑器简介

WeChatFormatter 以 Markdown 为输入，一键渲染为带**内联样式**的公众号 HTML，粘贴后样式不易丢失。

核心亮点：

- 本地一键排版，默认无需联网；可选手动配置 API 做 AI 润色 / 扩写 / 翻译
- 内置多套公众号主题（摸鱼绿、红白、石墨极简等）与 CSS 预设，一键切换
- 「复制到公众号」自动烘焙样式，兼容公众号编辑器结构
- 草稿自动保存、多草稿管理、历史版本回溯
- 支持导出 Markdown / HTML / 长图 / 纯文本
- 预览区可直接微调，支持撤销 / 重做、搜索替换、手机预览

本地启动（在仓库根目录）：

```bash
npm start
# 打开 http://localhost:3000/WeChatFormatter/
```

## 一、Markdown 基础语法

### 标题

在文本前添加 `#` 设置层级，`#` 越多层级越低：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

也支持部分中文标题习惯写法，例如 `一、标题`、`（一）标题`、`【标题】`。

小提示：若公众号后台已单独填写标题，正文建议从二级标题写起，避免重复出两个大标题。

### 列表

- 无序列表：`- 项` 或 `* 项`（支持嵌套缩进）
- 有序列表：`1. 项`
- 任务列表：`- [ ] 待办` / `- [x] 已完成`

### 文字样式

| 样式 | 语法 | 说明 |
|------|------|------|
| 粗体 | `**文字**` / `__文字__` | 加粗 |
| 斜体 | `*文字*` / `_文字_` | 斜体 |
| 删除线 | `~~文字~~` | 灰色删除线 |
| 高亮 | `==文字==` | 黄色背景高亮 |
| 文字颜色 | `{#E53E3E}文字{/#}` | 色值着色 |
| 中文色名 | `{红}文字{/}` | 便捷色名 |
| 颜色嵌套 | `{#F90606}外层{#0643F9}内层{/#}继续{/#}` | 支持嵌套 |

工具栏也提供 **B / I / 着色 / 高亮** 等快捷按钮，选中文字后点击即可包裹。

### 链接

```markdown
[文字](https://example.com)
```

### 引用块

```markdown
> 这是一段引用
```

预览中为浅蓝左边框 + 浅灰底样式。

### 表格

```markdown
| 列1 | 列2 |
| --- | --- |
| A   | B   |
```

支持斑马纹与圆角表头；可从 Excel 直接粘贴单元格，自动转为 Markdown 表格。表头背景色可在工具栏「表头」取色器调整。

### 分割线

单独一行写三个短横线：

```markdown
---
```

注意上方需空一行，避免被解析成标题。

### 图片插入

```markdown
![说明](图片地址)
```

也可以：

- 从剪贴板粘贴图片 → 自动插入 `![](pasted:N)` 并渲染
- 工具栏「图片 / 图」快捷插入
- 拖入或导入 `.md` 文件时一并保留图片引用

导出 Markdown 时，粘贴图可转为 base64，方便带走。

## 二、排版与高级功能

### 一键排版

点击 **「一键排版」** 或按 `Ctrl+S`：

1. 将左侧 Markdown 渲染为带内联样式的预览 HTML  
2. 自动注入字数 / 段落 / 预计阅读时间  
3. 应用首行缩进、字体、段间距与当前主题  
4. 错别字修正与敏感词检测（有风险时会提示）

默认走**本地排版**；在页头「API 配置」填写 Key 后，可使用 AI 助手（润色 / 扩写等）。

![一键格式化](../README.assets/image-20260617175157155.png)

### 复制到公众号

排版完成后点 **「复制到公众号」**：

- 自动补全公众号所需结构（如 `<span leaf="">`）
- 烘焙计算后的内联样式
- `div` 转换为更兼容的 `section`

然后到公众号后台 `Ctrl/⌘ + V` 粘贴即可。也可使用「纯文本」只复制文字。

### 样式模板与 CSS

工具栏「样式模板」可一键切换主题，并同步标题色、字体、段距与 CSS：

| 类型 | 模板示例 |
|------|----------|
| 公众号主题 | 摸鱼绿、红白色系、石墨极简、留白禅意、摸鱼票据、橄榄手记 |
| 其他风格 | ColorUI、科技感、中华古典、水墨留白、赛博霓虹、暗黑风格 |

需要精细控制时打开 **CSS** 面板：可从更多配色预设起步，保存后下次排版自动生效。

标题 H1～H4 颜色/字号、字体（系统 / 衬线 / 等宽 / 楷体 / 宋体）、段间距、首行缩进均可在工具栏实时调整。

### 代码

- 行内代码：`` `code` ``
- 代码块：三个反引号 + 语言名

```js
console.log('Hello WeChatFormatter')
```

### 数学公式

使用块级公式：

```markdown
$$ E = mc^2 $$
```

由 CodeCogs 渲染为 PNG；复制到公众号时会转为 base64，避免外链/SVG 插入失败。

### 目录生成

点击工具栏 **「目录」**，根据预览中的 H1～H4 在文章顶部生成可点击目录。

![添加目录](../README.assets/image-20260617175218499.png)

### 预览区编辑

预览区支持直接改字（WYSIWYG）：

- `Ctrl+Z` / `Ctrl+Y` 撤销、重做（最多约 50 步）
- `Ctrl+F` 搜索 / 替换
- 页眉「手机预览」可将预览宽度限制为约 375px

### AI 助手（可选）

配置 API Key 后，点击 **AI 助手**：

| 能力 | 说明 |
|------|------|
| 润色 | 优化表达、修正语病 |
| 扩写 | 丰富细节 |
| 缩写 | 精简提炼 |
| 翻译 | 中译英 / 日 / 韩 |

结果可写回编辑区，再重新一键排版。

### 草稿、历史与导出

| 功能 | 说明 |
|------|------|
| 自动草稿 | 输入防抖保存到浏览器 localStorage |
| 多草稿 | 「草稿」面板保存 / 切换 / 删除 |
| 历史版本 | 每次排版可留快照，最多约 20 条可回溯 |
| 导出 Markdown | 下载 `.md` |
| 导出 HTML | 独立网页，可本地打开 |
| 导出长图 | html2canvas 渲染预览为 PNG |
| 界面主题 | 深色 / 浅色，偏好会记住 |
| 全屏编辑 | 页眉按钮进入专注模式 |

首次打开会弹出使用帮助；之后可点页眉 **「帮助」**，或按 `?` 查看快捷键。
