02 Products

两个独立产品，一套写作链路

从草稿到排版，全部跑在浏览器本地。可一起用，也可分别部署。

mdEditor

Markdown 编辑器

左侧写作、右侧实时预览。支持公式、Mermaid、多格式导出与 10 种界面语言。内容自动保存在本地，断网也能继续。

  • 拖拽导入 · 双栏 / 单栏布局 · 源码模式
  • 导出 MD / HTML / Word / PDF / 长图
  • KaTeX 公式 · Mermaid 图表 · 深浅色主题
  • 依赖已本地 vendor，可完全离线使用
立即使用 mdEditor

WeChatFormatter

公众号排版工具

把 Markdown 一键变成可粘贴公众号后台的内联样式 HTML。内置多套主题模板，支持预览微调、草稿管理与长图导出。

  • 本地一键排版 · 复制到公众号不丢样式
  • 摸鱼绿 / 红白 / 石墨极简 等多套模板
  • 粘贴图片与 Excel · 自定义 CSS · 目录生成
  • 可选 AI 润色 / 扩写 / 翻译（需 API Key）
立即使用 WeChatFormatter

03 Principles

AI 时代，Markdown 是第一语言

工具应该足够轻：打开浏览器，输入文字，看到排版，然后把它带走。

01 / LOCAL-FIRST

数据只在本地

草稿与偏好写入浏览器存储，不强制账号，不默认云同步。

02 / ZERO-INSTALL

即开即用

纯前端静态资源。双击 HTML，或 npm start 一键托管。

03 / AGENT-READY

为 Agent 而生

Markdown 是 Agent 的通用输出。写完即可预览、导出，或送进公众号排版。

开始写，或开始排版

两个项目互不依赖，可独立部署到任意静态托管。
